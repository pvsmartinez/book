## The Core Shift
Based on the new definition (`Value Function = Rules + Track + Competitors`), Part III should stop being a list of "Filter Examples" (AI, Schools, Markets) and start being a course on the **Anatomy of the Value Function**.

Each chapter should teach one variable of the equation, using the examples (AI, Schools, etc.) as case studies.

## Detailed Chapter Outline

### Chapter 09: The Invisible Judge (The Equation)
*   **Refactor Goal:** Define the term "Value Function" rigorously.
*   **The Hook:** Why does the "Best" product fail? Why does the "Smartest" student fail?
*   **The Reveal:** Because "Best" and "Smart" are subjective. The System only cares about "Fit."
*   **The Formula:** `Value Function = Rules + Track + Competitors`.
*   **The Metaphor:** The "Judge" is not a person. It is the **Mold**. The runner pours themselves into the mold to fit.

### Chapter 10: The Rule Set (Explicit Constraints)
*   **Concept:** "The Rule" is the codified win condition.
*   **Key Insight:** If you write a rule, the system will optimize for the *letter* of the rule, not the *spirit*.
*   **Case Study (AI):** The AI that paused the game to "not lose." It followed the rule perfectly but missed the point.
*   **Case Study (Bureaucracy):** Compliance vs. Competence.

### Chapter 11: The Competitors (Relative Fitness)
*   **Concept:** Value is not absolute; it is relative to the other runners.
*   **Key Insight:** "Good Enough" yesterday is "Failure" today if the competitors improved. The Value Function serves to filter the *population*, not just the individual.
*   **Case Study (The Market):** The Invisible Hand isn't guiding; it's selecting based on the current field. If all bakers are using cheap flour, you must too (The "Race to the Bottom" mechanics).
*   **Case Study (Sports):** The meta.

### Chapter 12: The Track (Implicit Constraints)
*   **Concept:** The Rules say "Run Fast," but the Track says "It's Muddy."
*   **Key Insight:** The *Medium* allows certain behaviors to thrive and kills others. You cannot build a delicate machine for a rough environment.
*   **The Shift:** This is where we move `13_the_medium_is_the_filter.md`.
*   **Case Study (Media):** Why short-form content won over blogs. Not because of a "Rule," but because of the "Track" (Mobile phones, attention spans).

### Chapter 13: The Proxy (The Map provides the Wrong Information)
*   **Concept:** We can rarely measure the thing we care about (Intelligence, Health), so we measure a Proxy (Grades, BMI).
*   **Key Insight:** The System optimizes for the Proxy, often destroying the Goal. (Goodhart's Law).
*   **Case Study (The Exam Trap):** This moves here. The Exam is the Proxy for Potential.

### Chapter 14: Synthesis (The Mold)
*   **Concept:** Recapping Part III. "You are what you measure."
*   **Key Insight:** Adaptation is indifferent. It will turn a Wolf into a Pug if the Value Function asks for a Pug.
*   **Transition to Part IV:** Now that we know *what* shapes us (The VF), let's look at what happens when that shape compounds over time.

### Chapter 14b: Interlude (The Audit)
*   **Exercise:** "Who is judging you?"
*   **Task:** Identify the Rules, Track, and Competitors in your own career/life.
